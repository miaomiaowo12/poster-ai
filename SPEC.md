# AI海报生成器 - 项目规范

## 1. 项目概述

- **项目名称**: PosterAI - AI智能海报生成器
- **项目类型**: 移动端优先的Web应用
- **核心功能**: 用户输入海报描述，AI生成海报，支持编辑和下载
- **目标用户**: 需要快速生成海报的普通用户、设计师、小商家

## 2. UI/UX 规范

### 2.1 布局结构

**页面组成**:
- 顶部导航栏 (56px)
- 主内容区域 (可滚动)
- 底部操作栏 (用于生成/保存操作)

**响应式断点**:
- 移动端: < 768px (主要设计目标)
- 平板: 768px - 1024px
- 桌面: > 1024px

### 2.2 视觉设计

**色彩方案**:
- 主色: #6366F1 (靛蓝色)
- 次色: #8B5CF6 (紫色)
- 强调色: #F59E0B (琥珀色)
- 背景色: #0F0F23 (深空蓝黑)
- 卡片背景: #1A1A2E (深紫蓝)
- 文字主色: #FFFFFF
- 文字次色: #A1A1AA (灰色)
- 成功色: #10B981
- 错误色: #EF4444

**字体**:
- 主字体: "Noto Sans SC", sans-serif
- 标题: 24px (bold)
- 副标题: 18px (semibold)
- 正文: 14px (regular)
- 小字: 12px

**间距系统**:
- xs: 4px
- sm: 8px
- md: 16px
- lg: 24px
- xl: 32px

**视觉效果**:
- 卡片圆角: 16px
- 按钮圆角: 12px
- 阴影: 0 4px 24px rgba(99, 102, 241, 0.15)
- 渐变背景: linear-gradient(135deg, #0F0F23 0%, #1A1A2E 100%)
- 玻璃态效果: backdrop-filter: blur(12px)

### 2.3 组件设计

**1. 顶部导航栏**
- 左侧: Logo + 应用名称
- 右侧: 历史记录按钮
- 背景: 半透明深色 + 毛玻璃效果

**2. 输入区域**
- 海报描述文本框 (多行，支持placeholder)
- 风格选择器 (下拉/标签选择)
- 尺寸选择器 (方形/竖版/横版)
- 生成按钮 (主按钮，渐变背景)

**3. 海报展示区**
- 海报卡片 (带圆角和阴影)
- 加载状态 (骨架屏 + 动画)
- 错误状态 (提示信息)

**4. 编辑工具栏**
- 位置: 固定在海报下方
- 工具: 文字编辑、滤镜、调整尺寸
- 操作: 重新生成、保存、下载

**5. 底部操作栏**
- 生成按钮 (主CTA)
- 保存到相册
- 分享按钮

### 2.4 交互设计

**动画效果**:
- 按钮悬停: scale(1.02) + 亮度提升
- 卡片出现: fadeIn + slideUp (300ms)
- 生成中: 脉冲动画 + 进度指示
- 页面切换: 平滑过渡

**反馈**:
- 点击反馈: 轻微缩放
- 加载反馈: 骨架屏
- 成功反馈: 绿色提示 + 震动(移动端)
- 错误反馈: 红色提示 + 重试选项

## 3. 功能规范

### 3.1 核心功能

**海报生成**:
- 用户输入描述文本
- 选择海报风格 (现代、简约、复古、赛博朋克等)
- 选择尺寸比例 (1:1, 9:16, 16:9)
- 调用AI API生成海报
- 显示生成结果

**海报编辑**:
- 文字编辑: 修改海报上的文字
- 滤镜效果: 应用不同滤镜
- 亮度/对比度调整
- 重新生成相似海报

**保存下载**:
- 保存到浏览器本地存储
- 下载为PNG/JPG格式
- 复制到剪贴板

### 3.2 AI集成

- 使用 Picsart AI 或 类似免费API
- 备用方案: 使用 Canvas + 预设模板生成
- 支持多次生成直到满意

### 3.3 数据管理

- 本地存储历史记录
- 支持清空历史
- 最多保存20条历史记录

## 4. 技术栈

- **框架**: React 18 + Vite
- **样式**: CSS Modules / Tailwind CSS
- **状态管理**: React Hooks
- **AI API**: Picsart AI (免费版) / Pollinations AI
- **图片处理**: html2canvas
- **动画**: Framer Motion
